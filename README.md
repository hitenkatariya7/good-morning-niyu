# good-morning-niyu

A little 3D morning room, built for Niyu.

## Updating content nightly
Everything that changes day-to-day lives in `content.json` — no coding needed:
- `note.title` / `note.body` — the note inside the jar
- `photos[0].src` — path to today's photo (drop new images in `photos/`)
- `photos[0].caption` — caption under the photo
- `dayCount` — the number shown when the mug is tapped
- `weatherMessages` — the sticky note text per weather condition (auto-selected each morning)
